# Bookshelf API

Backend for Bookshelf project

## Prerequisites

```bash
$ ruby --version
$ rails --version
```

## Development

```bash
$ git clone https://github.com/tediore-wf/bookshelf-api.git
$ cd bookshelf-api
$ bundle install
$ bundle exec rake db:create
$ bundle exec rake db:migrate
$ bundle exec rails s
```
